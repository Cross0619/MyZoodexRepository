Android/Kotlin 初学者向け

動物図鑑アプリ「MyZoodex」で学ぶ Android 入門

このサイトは、既存の Android アプリ「MyZoodex」を題材に、Android Studio のセットアップから Kotlin・Jetpack Compose の基本、サンプルコードの読み方までを丁寧に解説する技術ブログです。まずは Android Studio をインストールし、少しずつコードに触れていきましょう。

想定読者: Android/Kotlin はほぼ未経験。まずは 1 本のアプリを自作してみたい方。

学習ステップの流れ

  1. Android Studio をインストールする
  2. Kotlin の基本文法を押さえる
  3. Jetpack Compose で UI を作る
  4. MyZoodex のコードを読み解く
  5. 公式ドキュメントで知識を広げる
