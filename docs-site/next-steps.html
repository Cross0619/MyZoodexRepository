次に進むための公式リソース

MyZoodex のコードをひととおり追い終えたら、公式のチュートリアルやリファレンスでより深く学びましょう。ここではおすすめの順番と対象読者の目安を紹介します。

Android Basics with Compose

初めて Android アプリを作る人向けの段階的コースです。UI だけでなく、アーキテクチャやデータ永続化、テストまで扱います。MyZoodex で学んだ Compose の基礎をさらに実践形式で固めたい人に最適です。

Jetpack Compose 公式チュートリアル

Compose の最新 API を体系的に試せるチュートリアルです。公式サンプル「JetNews」や「Now in Android」を通して、より複雑な UI パターンや状態管理を学べます。UI 設計の引き出しを増やしたいときに。

Kotlin 公式ドキュメント

言語の仕様を詳しく知りたい場合は、Kotlin 公式の言語リファレンスが頼りになります。コレクション操作やコルーチンなど、MyZoodex には登場しない機能も幅広くカバーされているので、必要に応じて逆引き的に活用しましょう。

読み進める順番の例

ステップ 対象 狙い
1. Android Basics with Compose UI も含めて 1 本作りたい人 アプリ全体の流れと Compose の実践
2. Jetpack Compose チュートリアル Compose の幅を広げたい人 複雑な UI / 状態管理のパターンを吸収
3. Kotlin 言語リファレンス 文法を深く理解したい人 コレクション、高階関数、コルーチンなどを体系的に学ぶ

最後に

公式ドキュメントは更新頻度が高いため、必ず最新版を確認しましょう。MyZoodex で掴んだ基礎の上に、自分だけのアプリを積み上げてください。
